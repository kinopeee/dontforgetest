# BugBot Configuration

## Response Language

**Language**: Japanese (日本語)

All BugBot responses, comments, and feedback on this repository must be written in Japanese. This includes:

- Bug descriptions and explanations
- Code review comments
- Suggested fixes and recommendations
- All explanatory text

Do not respond in English. Always use Japanese for all communications.

# BugBot の設定

## 応答言語

このリポジトリにおけるすべての BugBot の応答、コメント、フィードバックは日本語で記述してください。これには以下が含まれます：

- バグの説明と解説
- コードレビューのコメント
- 修正案と推奨事項
- すべての説明文

英語での応答は行わないでください。すべてのコミュニケーションで日本語を使用してください。
