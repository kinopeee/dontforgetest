# Code of Conduct

We are committed to providing a safe, welcoming, and constructive environment for everyone participating in this project.

## Expected behavior

- Be respectful and considerate in communication
- Give constructive feedback
- Respect differences in backgrounds and experience
- Focus on solving problems; avoid personal attacks

## Unacceptable behavior

- Harassment, discrimination, insults, threats, stalking
- Sexualized language or imagery, or other inappropriate content
- Sharing personal information without consent
- Disruptive behavior (spam, trolling, etc.)

## Scope

This Code of Conduct applies to all project spaces, including Issues, Pull Requests, Discussions, and other project-related communication.

## Reporting

To report a violation, contact:

- [@kinopee_ai on X](https://x.com/kinopee_ai)

## Enforcement

We will review reports and may take action as appropriate, including (but not limited to):

- Warnings
- Editing/removing content
- Locking Issues/PRs
- Temporary or permanent restrictions (e.g., blocking)

# 行動規範（Code of Conduct）

このプロジェクトは、すべての参加者にとって安全で建設的な場であることを目指します。

## 期待する行動

- 相手を尊重し、敬意をもってコミュニケーションする
- 建設的なフィードバックを心がける
- 立場や経験の違いを尊重する
- 問題解決に焦点を当て、個人攻撃をしない

## 許容しない行動

- 侮辱、差別、嫌がらせ、脅迫、ストーキング等
- 性的な表現を含む不適切な発言や画像の投稿
- 個人情報の公開（本人の同意がないもの）
- プロジェクトの運営を妨害する行為（スパム、荒らし等）

## 適用範囲

本行動規範は、Issue / Pull Request / Discussion / その他プロジェクトに関連するコミュニケーション全般に適用されます。

## 連絡先

行動規範違反の報告は、以下へご連絡ください。

- [@kinopee_ai on X](https://x.com/kinopee_ai)

## 執行（Enforcement）

報告内容を確認し、必要に応じて以下の対応を行います（状況により順不同）。

- 注意喚起
- コメントの編集/削除
- Issue/PR のロック
- 一時的または恒久的な参加制限（ブロック等）
