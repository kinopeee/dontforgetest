# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability, **please do not open a public Issue**.
Contact the maintainer instead:

- Contact: [@kinopee_ai on X](https://x.com/kinopee_ai)

If possible, include:

- Steps to reproduce
- Impact (what an attacker can do)
- Affected versions
- Relevant logs/screenshots (without secrets)

---

# セキュリティポリシー

## 脆弱性の報告

脆弱性と思われる問題を見つけた場合は、**公開 Issue には書かず**、以下の連絡先へご連絡ください。

- 連絡先: [@kinopee_ai on X](https://x.com/kinopee_ai)

可能であれば、次の情報を含めてください。

- 再現手順
- 影響範囲（何ができてしまうか）
- 対象バージョン
- 参考ログ/スクリーンショット（機密を含まない範囲）
